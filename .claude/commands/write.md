# 블로그 작성 (모드 B)

여러 참고 블로그를 종합하여 새 블로그 포스트를 작성합니다.

## 작업 순서

1. `drafts/$ARGUMENTS/` 폴더에서 참고 자료 확인
   - `refs.md`: 참고 URL 목록 읽기
   - `notes.md`: 작성자 메모 읽기 (있으면)
2. refs.md의 URL들을 fetch하여 핵심 내용 분석
3. 여러 소스의 내용을 종합하여 재구성 (단순 복사 금지)
4. 빠진 내용이나 최신 정보는 웹 검색으로 보충
5. 적절한 카테고리(대분류/소분류) 판단
6. `_posts/` 하위 폴더 구조를 확인하여 올바른 경로 결정
7. CLAUDE.md 규칙에 맞게 포스트 작성:
   - front matter 생성
   - ##부터 시작하는 이모지 헤더
   - 실무 관점 코드 예제 포함
   - 하단 📚 참고에 모든 참고 URL 정리
8. `_posts/[대분류]/[소분류]/YYYY-MM-DD-slug.md` 로 파일 생성
9. 생성한 파일의 경로를 안내
