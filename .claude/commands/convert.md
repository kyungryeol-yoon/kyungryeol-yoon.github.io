# 블로그 변환 (모드 A)

기존 md 파일을 블로그 규칙에 맞게 변환합니다.

## 작업 순서

1. `drafts/` 에서 변환할 md 파일 읽기: $ARGUMENTS
2. 내용을 분석하여 적절한 카테고리(대분류/소분류) 판단
3. `_posts/` 하위 폴더 구조를 확인하여 올바른 경로 결정
4. CLAUDE.md 규칙에 맞게 변환:
   - front matter 생성 (title, date, categories, tags)
   - 헤더를 ##부터 시작하도록 변경
   - 이모지 섹션 헤더 적용
   - 존댓말 통일
   - 코드블록 언어 태그 확인/추가
   - 하단에 📚 참고 섹션 추가
5. `_posts/[대분류]/[소분류]/YYYY-MM-DD-slug.md` 로 파일 생성
6. 생성한 파일의 경로를 안내

**git 작업은 하지 않음**
