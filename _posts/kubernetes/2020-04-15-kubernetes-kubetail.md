- "kubectl logs -f "를 실행하는 것과 동일하지만 여러 Pod에 적용

### Install 
```
git clone https://github.com/johanhaleby/kubetail.git
```

> kubetail에 대한 설정 방법은 [설치 문서](https://github.com/johanhaleby/kubetail)를 참고하시기 바랍니다.
